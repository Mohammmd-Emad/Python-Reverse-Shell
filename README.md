# Python Reverse Shell – Cybersecurity Educational Project

This project is a simple reverse shell implemented in Python for educational purposes and cybersecurity learning.  
The goal of this project is to understand how remote command execution works and how reverse shells are used in penetration testing and security research.

## ⚠ Disclaimer

This project was created for educational purposes only.
It is intended to be used in controlled environments such as virtual machines or lab setups.
Do NOT use this code on systems without permission.

## Features

- Reverse TCP connection
- Remote command execution
- Client / Server communication using sockets
- Works on local network / lab environment
- Written in Python

## What I learned

- Python socket programming
- TCP/IP networking basics
- How reverse shells work
- Basic penetration testing concepts
- Antivirus detection testing using VirusTotal

## Testing

The project was tested in a controlled environment and uploaded to VirusTotal for analysis to understand how security systems detect suspicious behavior.
